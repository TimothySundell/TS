# Extract the mean and median values of Ig-genes from a list of PCA feature loadings

Takes as input the output from
[TS_extract_ranked_genes_per_PC](https://timothysundell.github.io/TS/reference/TS_extract_ranked_genes_per_PC.md)

## Usage

``` r
TS_extract_ig_mean_median_rank_per_PC(PCA_loadings)
```

## Arguments

- PCA_loadings:

  List, the output from
  [TS_extract_ranked_genes_per_PC](https://timothysundell.github.io/TS/reference/TS_extract_ranked_genes_per_PC.md)
