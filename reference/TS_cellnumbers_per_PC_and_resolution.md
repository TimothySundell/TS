# Calculate cellnumbers per cluster for each combination of 'number or PCs' and 'resolution'

Calculate cellnumbers per cluster for each combination of 'number or
PCs' and 'resolution'

## Usage

``` r
TS_cellnumbers_per_PC_and_resolution(seurat_object, PCs)
```

## Arguments

- seurat_object:

  Input seurat object

- PCs:

  Number of maximum PC you want to look at
